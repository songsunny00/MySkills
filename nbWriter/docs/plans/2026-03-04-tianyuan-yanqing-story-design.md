# 田园现代言情·故事设计文档

**赛道**: 田园现代言情
**篇幅**: 40章，约60,000-80,000字
**基调**: 治愈·成长·现实·温暖
**创建日期**: 2026-03-04

---

## 一、核心故事

> 一个被AI浪潮裁掉的女程序员，带着劳动仲裁的等待和满身疲惫，因为一条"义工换食宿"的招募帖，来到一个陌生村子——然后用她以为已经没用了的技术，重新找到了自己存在的意义。

**核心主题**:
- 被AI取代的人，用AI帮助更多被时代抛下的人找到出路（最核心的反转）
- 躺平与向前之间，年轻人真正需要的是"缓冲地带"
- 城市与乡村互相照见，没有谁的生活更对

## 二、人物设定

### 女主：陈以码（占位名，可替换）
- **年龄**: 30岁
- **职业**: 后端工程师，6年工龄，无限期合同
- **被裁经过**: AI浪潮下公司大规模重组，单方面降薪30%变更合同→她拒签→被冷藏、剥夺项目→上裁员名单→公司拒付2N→申请劳动仲裁（等待开庭中）
- **心理状态**: 倦怠+愤怒+迷茫，不是软弱，是主动按下暂停键
- **来村原因**: 等待开庭的空白期，刷到"归田青年公寓"义工换食宿的帖子，冲动买票
- **性格**: 轴、有原则、不擅长示弱，但内心细腻；职业病严重（看到问题忍不住想解决）
- **核心技术产品**: **"归巢"——返乡青年画像与资源匹配工具**
  - 入住青年填写技能/需求，系统自动匹配
  - 设计师对接有农产品找销路的村民
  - 程序员帮村里做数字化小工具
  - 手艺人找到传承对象
  - 把民宿变成真正的"人才交换所"
- **成长弧**: 从"我的技术被AI取代了" → "我的技术让人找到彼此"

### 男主：沈归田（占位名，可替换）
- **年龄**: 32岁
- **职业**: 前互联网产品经理
- **来村原因**: 两年前因坚持用户优先被资本方架空出局，用赔偿金租下村里老宅做民宿，比女主先落地
- **民宿定位**: **"归田青年公寓"**——面向返乡/疗愈青年，偏青年公寓风格，非传统农家乐
- **性格**: 不急、不催、用行动说话；表面随和，内有棱角
- **关键设定**: 女主一开始以为他是土生土长的本地人，直到合作中发现他对产品逻辑的专业理解
- **与女主的化学反应**: 他已落地，她还在坠落——节奏差是感情慢热的天然土壤
- **身份揭秘时机**: 约15-17章，深夜改bug，男主说出专业术语被追问，讲出自己的故事，女主第一次在他面前哭出来

### 林杳（城市闺蜜）
- **年龄**: 29岁
- **职业**: 前互联网产品运营
- **困境**: 长期高压→轻度抑郁，正在休病假
- **登场时机**: 约6-8章，担心女主独自在外，突然出现
- **功能**: 友情线主要载体；心理健康议题的具体呈现；她的好转历程是副线之一
- **结局**: 回城，但装了"归巢"APP，"我在城里给你们拉流量"

### 谷小满（本地女孩）
- **年龄**: 22岁
- **身份**: 土生土长的村里人，没出过远门
- **性格**: 开朗直接，对外面世界充满好奇，有点莽
- **与女主的化学反应**: 女主羡慕她的简单自在，她羡慕女主见过大世面；"围城"式互相照见
- **关键冲突**: 约19-21章，她想去城市闯，女主支持还是劝阻？两人第一次真正争吵
- **结局**: 通过"归巢"对接到城市的远程工作机会，不用离开村子，城市来找她

### 父母（配角）
- **家庭背景**: 普通工薪家庭，父亲退休工人，母亲做过小生意，生活不宽裕但知足
- **性格**: 母亲主导、嘴上唠叨心里清醒眼睛毒；父亲话少但一针见血
- **来访弧线**（约33-36章）:
  1. **第一天**: 带着相亲对象二维码来拉人回城找工作找对象
  2. **第二天**: 母亲偷偷考察男主，父亲吃了他做的饭筷子没停
  3. **第三天**: 王叔儿子二维码悄悄收起来，开始撮合
  4. **离开前**: 父亲单独找男主，"她倔的事从来没错过，你懂我意思吧"

### 流动过客（4-5位，分散在全文）
| 出场时机 | 人物类型 | 代表困境 |
|---------|---------|---------|
| 约9-10章 | 考公三连败应届生 | 躺平还是继续卷 |
| 约20章 | 被降薪的中年互联网人 | 35岁职场困境 |
| 约25章 | 辞职gap year的设计师 | 自我价值怀疑 |
| 约28章 | 远程办公的自由职业者 | 自由背后的孤独 |

## 三、40章结构

### 序章·城市（1-3章）
**基调**: 快节奏，新闻感，交代背景，不渲染苦情

| 章节 | 内容 |
|------|------|
| 第1章 | AI裁员浪潮背景，公司推行合同变更，陈以码拒签 |
| 第2章 | 被冷藏→上名单→公司拒付2N→申请仲裁 |
| 第3章 | 等待开庭的空白期，刷到义工帖，冲动买票 |

### 第一幕·落地（4-12章）
**基调**: 手忙脚乱，笑中带泪，新友情萌芽

| 章节范围 | 内容 |
|---------|------|
| 4-5章 | 抵达，义工生活开始，遇见谷小满，误以为男主是本地人 |
| 6-8章 | 林杳突然来访，三人女生组合成型，林杳状态初现端倪 |
| 9-10章 | 第一批流动过客入住，"归巢"概念种子萌芽 |
| 11-12章 | 林杳状态崩溃一次，女主第一次直面抑郁症，意识到心理健康不是矫情 |

**仲裁线**: 偶尔弹出律师消息，女主刻意不看

### 第二幕·生根（13-30章）
**基调**: 事业线展开，感情升温，群像丰满

| 章节范围 | 内容 |
|---------|------|
| 13-15章 | 男主邀请合作，女主介入运营，职业病全面爆发 |
| 16-18章 | **【身份揭秘】** 深夜改bug，男主讲出自己的故事，女主第一次哭出来 |
| 19-21章 | 谷小满想去城市，女主支持还是劝阻，第一次真正争吵后和解 |
| 22-24章 | "归巢"正式上线，第一批匹配成功，民宿开始真正发挥"缓冲地带"价值 |
| 25-27章 | 流动过客故事高峰期，各种困境涌入，感情线克制升温 |
| 28-30章 | 林杳开始好转，决定留下做"归巢"运营；谷小满通过"归巢"接到远程工作 |

**仲裁线**: 约22章，收到开庭通知，她第一次真正动摇——是否回去

### 第三幕·选择（31-40章）
**基调**: 危机→破防→清醒→选择→圆满

| 章节范围 | 内容 |
|---------|------|
| 31-32章 | 仲裁胜诉，2N赔偿到账，她反应是恍惚而非喜悦 |
| 33-36章 | **【父母来访】** 完整四阶段弧线（拉人→考察→撮合→父亲那句话） |
| 37-38章 | 女主正式决定留下，用赔偿金注册工作室，和男主合伙升级民宿 |
| 39章 | 谷小满："你们城里人跑来找的，就是我们一直觉得没什么的。" |
| 40章 | **【结局】** 林杳回城装了APP；谷小满第一笔远程收入到账；女主院子里写代码，男主送汤，什么都没说，都够了 |

**最后一幕背景音**: 国家整治AI裁员乱象的新闻在手机上一闪而过，她看了一眼，放下手机，继续写代码。

## 四、关键议题与情节钩子

### 心理健康议题
- 林杳的轻度抑郁：不是崩溃式呈现，而是"撑着撑着突然撑不住"的真实感
- 女主的职业倦怠：不是软弱，是系统性压力下的正常反应
- 流动过客的各种状态：焦虑、躺平、迷失、自我怀疑

### AI与技术议题
- 反讽设计：被AI裁掉的人，用AI工具帮助更多被时代抛下的人
- "归巢"的技术门槛：不高精尖，但真实有用
- 仲裁背景：劳动权益保障，普法意义

### 城乡互照议题
- 城市人来乡村找"简单"，乡村人向往城市的"可能"
- 没有谁的生活更对，只有适不适合自己
- 谷小满的结局：不是逃离，是在地获得更大的可能

## 五、模板使用说明

本故事设计对应以下模板，生成时应将此文档作为背景知识注入：

- `worldbuilding.jinja2`：生成村落背景，需体现青年公寓客群特点
- `character_creation.jinja2`：按上述人物卡生成详细档案
- `plot_design.jinja2`：基于三幕结构展开，注意仲裁线的城乡张力
- `outline_generation.jinja2`：40章，注意流动过客穿插节奏
- `chapter_generation.jinja2`：注意序章快节奏与乡村章慢节奏的切换
